# Task Manager

A modern and user-friendly Task Manager application that helps users organize, track, and manage their daily tasks efficiently.

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Filter tasks by status
* Responsive and clean user interface
* RESTful API built with Express.js
* Efficient task management workflow

## Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/24A31A0524/Task-Manager.git
```

### 2. Navigate to the Project Directory

```bash
cd Task-Manager
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start the Application

```bash
npm start
```

### 5. Open in Browser

Visit:

```text
http://localhost:5176
```

## Usage

* Create tasks to organize your work.
* Update task details whenever needed.
* Mark tasks as completed.
* Delete tasks that are no longer required.
* Filter tasks based on their status.
