# Task Manager

A modern and user-friendly Task Manager application that helps users organize, track, and manage their daily tasks efficiently.

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Filter tasks by status
* Responsive and clean user interface
* Persistent data storage (if implemented)

## Technologies Used

* HTML
* CSS
* JavaScript
* React (if applicable)
* Node.js / Express (if applicable)
## Installation

1. Clone the repository:

```bash
git clone https://github.com/24A31A0524/Task-Manager.git
```

2. Navigate to the project directory:

```bash
cd Task-Manager
```

3. Install dependencies:

```bash
npm install
```

4. Start the application:

```bash
npm start
```

## Usage

* Create tasks to keep track of your work.
* Update task details whenever needed.
* Mark tasks as completed once finished.
* Delete tasks that are no longer required.
